Invention du langage javascript

Les Variables

Une variable est utilisé pour stocker des données. Ce sont en quelques sorte des tiroirs dans lesquels nous stockons des données de types divers (nombre, chaîne de caractère, object…) En javascript on utilise le mots clés var pour déclarer une variable.

var a; // nous déclaration la variable "a", mais sans valeur pour le moment (vide)

On utilse l'opérateur d'assignation = pour assigner une valeur à une variable.

var a; // nous déclaration la variable "a", mais sans valeur pour le moment (vide)
a = 12; // assignation: nous donnons une valeur à la variable précédemment déclarée

On peut déclarer et assigner une variable en une seul instruction:

var a = 12; // déclaration + assignation

On peut additionner/multiplier/soustraire etc… des variables entre elles:

var a = 3;
var b = 2;
var c = a + b; // = 5

var foo = "foo";
var bar = "bar";
var baz = foo + bar; // "foobar"
