Invention du langage javascript

Les types de variables

En programmation nous pouvons utiliser des variables plus complexe que de simple nombre. En Javascript il existe différents types de variables.

Les types les plus courants sont:

  • Numbers
    • Float: un nombre, tel que 1.21323, 4, -33.5, 100004 ou 0.123
    • Integer: un nombre tel que 1, 12, -33, 140 mais pas 1.233

exemple: var height = 1.78;, var age = 12;

  • String: une ligne de texte tel que "pony", "unicorn" or "My little pony"

exemple: var text = &quot;Hello world&quot;;

  • Boolean: soit true (vrai) or false (faux), et rien d'autre.

exemple: var foo = true; ou `var foo = false;

  • Arrays: une collection de valeurs comme: 1,2,3,4,'hello world'

exemple: var tableau = [1, 3, 2, 12, &quot;hello&quot;];

  • Objects: une représentation d'un objet plus complexe

exemple:

var obj = {
    name: "bob",
    age: 32,
    height: 1.78,
    text: "Hello I'm bob…"
}
  • null: une variable qui contient "null" ne contient pas de données valide (ne contient de de Number, String, Boolean, Array ou Object valide) ou est volontaire assigner à vide.

exemple: var a = null;

  • undefined: la valeur "undefined" est obtenue lorsqu'on appelle une propriété qui n'existe pas dans un object, ou lorsqu'une variable à été déclaré mais n'a pas de valeur assignée.

exemple:

var a;
alert(a); // "alert" permet d'afficher un message dans une petite fenêtre d'information

On dit que le Javascript n'a pas de typage fort, cela signifie que vous n'avez pas a déclarer explicitement le type d'une variable. Vous avez juste besoin d'utiliser le mot clé var pour indiquer que vous déclarez une variable et l'interpréteur trouvera de quel type est votre variable.
