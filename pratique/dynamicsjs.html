Invention du langage javascript

Dynamics JS

Dynamicsjs est une librairie JS (au même titre que jquery). Sauf que son utilité n'est pas de manipuler le DOM comme jquery mais de créer des animations un peu plus poussées.

Dynamicsjs permet de créer des animations plus "physique", c'est à dire avec des effets de gravités, de rebonds etc...

Le site internet de dynamicsjs http://dynamicsjs.com/

Utiliser dynamicsjs

Comme pour jquery, on peut soit télécharger les sources depuis son navigateur, ou alors utiliser la version CDN:

CDN : https://cdnjs.cloudflare.com/ajax/libs/dynamics.js/0.0.7/dynamics.min.js

dynamics.animate(el, properties, options)

Le coeur de dynamics est cette function .animate(), elle prend 3 arguments :

  • un node HTML
  • des propriétés sous forme d'object
  • des options sous forme d'object

Les propriétés

Les propriétés sont des propriétés CSS, la seul différence avec le CSS est qu'ici les propriétés qui prennent habituellement un tiret (background-color, margin-top, text-align, etc...) s'écrivent sans tiret de cette forme : backgroundColor, marginTop, textAlign etc...

exemple :

{
    marginTop: 20,
    backgroundColor: #fafafa,
    color: #000
}

Les options

Les options sont propre au dynamicsjs, elles permettent de régler votre animation, son type, sa durée etc...

voici la liste des options disponibles :

  • type : c'est le type d'animation: dynamics.spring, dynamics.easeInOut, dynamics.bounce, dynamics.forceWithGravity, dynamics.gravity, dynamics.easeInOut, dynamics.easeIn and dynamics.easeOut, dynamics.linear, dynamics.bezier (par defaut: dynamics.easeInOut)
  • frequency : la fréquence (physique)
  • friction : la friction (physique)
  • duration : la durée en millisecondes (defaut: 1000)
  • delay : le delais en millisecondes (default: 0)
  • complete (optionel) lorsque l'animation est finit je peux appeller une function.

exemple :

{
    type: dynamics.spring,
    frequency: 200,
    friction: 200,
    duration: 1500,
    delay: 400
}

Documentation complète:

https://github.com/michaelvillar/dynamics.js

Exemple tout simple

Créons une div:

<div class="circle"></div>

Nous créons un cercle en css :

.circle {
    width: 20px;
    height: 20px;
    border-radius: 100%;
    background-color: blue;
}

javascript avec dynamicsjs :

  • On sélectionne le cercle (.circle)
  • je créer deux functions: step1 et step2 : elles correspondent aux deux états de l'animation (aller/retour).
  • pour step1 : je bouge mon cercle vers la droite (150px), je change son opacité, et sont background-color
  • pour step2 : je bouge mon cercle à sa position inital (0px), je change sa taille (scale), je remet son opacité, et je change son background-color.
// je sélectionne mon cercle
var circle = $('.circle');

function step1() {
  dynamics.animate(circle[0], {
    translateX: 150,
    opacity: 0.5,
    backgroundColor: "#DE26A6"
  }, {
    type: dynamics.spring,
    frequency: 200,
    friction: 200,
    duration: 1500,
    complete: step2
  });
}

function step2() {
  dynamics.animate(circle[0], {
    translateX: 0,
    scale: 0.8,
    opacity: 1,
    backgroundColor: "#4120F3"
  }, {
    type: dynamics.spring,
    frequency: 200,
    friction: 200,
    duration: 1500,
    complete: step1
  });
}

step1(); // pour commencer j'appelle la function step1
