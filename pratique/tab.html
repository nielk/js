Invention du langage javascript

Tab (tabulation)

Dans cette section nous allons voir comment créer un menu type "tab"

1 - Création dossier/fichiers

  • Créer un dossier nommé "menu-01"
  • Ouvrez Sublime Text et ouvrez ce dossier
  • Dans Sublime Text, dans le dossier "menu-01" créez les fichiers suivant: index.html, main.css, main.js

2 - Chargement script/style

Nous allons définir le contenu de la page HTML, il nous faut:

  • une page HTML de bases (html, head, body etc...)
  • charger jquery via la balise script
  • charger via la balise script le fichier main.js
  • charger via la balise link le fichier main.css

4 - Le CSS

  • Nous allons placer le menu tab, en bas de la page comme sur une application mobile
/* on positionne en bas le menu tab */
.tab-nav {
  position: fixed;
  right: 0;
  bottom: 0;
  left: 0;
  height: 60px;
  background-color: #F5F5F5;
  text-align: center;
}

.tab-nav ul {
  padding-left: 0;
}

/* style des btn de nav tab */
.tab-btn {
  display: inline-block; /* alignement horizontale */
  list-style-type: none; /* pas de puce */
  cursor: pointer;
}
  • Nous allons créer les class relatives à nos panels
.panel {
    position: fixed;
    top: 0;
    right: 0;
    bottom: 60px;
    left: 0;
    overflow: scroll;
    padding: 30px;
    background-color: #cacaca;
    opacity: 0; /* je cache les panels par defaut */
    transition: .5s opacity ease; /* animation */
}

/* c'est cette classe qui rend visible un panel */
.panel-visible {
    opacity: 1;
    z-index: 1; /* je place le panel par dessus tout les autres */
    transition: .5s opacity ease; /* animation */
}

4 - Javascript

  • sélectionner les boutons du tab-nav
  • écouter les évènements click sur les boutons
  • au click: enlever la class .panel-visible de tout les panels
  • ajouter la class .panel-visible sur le panel que l'ont veut afficher
$(document).ready(function() {
  // je sélectionne les buttons de la tab-nav
  var btns = $(".tab-btn");

  // j'écoute les clicks sur les btns
  btns.on("click", function() {
    // je récupère la valeur de l'attribut data-panel
    // il contient le panel que je veux afficher
    var pannel = $(this).attr('data-panel');

    // je masque tout les panels en enlevant la class .panel-visible
    $(".panel").removeClass("panel-visible");
    // je rajoute la class .panel-visible uniquement 
    // sur le panel que je veux afficher
    $("." + pannel).addClass("panel-visible");
  });
});
