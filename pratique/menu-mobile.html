Invention du langage javascript

Menu

Dans cette section nous allons apprendre à créer un menu caché qui apparaît via un bouton.

1 - Création du dossier et fichiers

  • Créer un dossier nommé "menu-01"
  • Ouvrez Sublime Text et ouvrez ce dossier
  • Dans Sublime Text, dans le dossier "menu-01" créez les fichiers suivant: index.html, main.css, main.js

2 - Charger les fichiers

Nous allons définir le contenu de la page HTML, il nous faut:

  • une page HTML de bases (html, head, body etc...)
  • charger jquery via la balise script
  • charger via la balise script le fichier main.js
  • charger via la balise link le fichier main.css

3 - Definir l'HTML

Nous allons faire une page basique, avec un menu.

Il nous allons donc ajouter (dans cette ordre) :

  • Un menu : une balise nav avec la classe menu qui contient une balise ul contenant des balises li avec des balises a avec la classe menu-link:
<nav class="menu">
  <ul>
      <li>
        <a href="" class="menu-link">Home</a>
      </li>
      <li>
        <a href="" class="menu-link">About</a>
      </li>
      <li>
        <a href="" class="menu-link">CV</a>
      </li>
      <li>
        <a href="" class="menu-link">Contact</a>
      </li>
  </ul>
</nav>
  • un élément qui ouvrira le menu : une div portant la classe menu-btn
<div class="menu-btn">
    Menu
</div>
  • du contenu pour notre page :
<section>
  <h1>Ma page avec menu</h1>
  <p>
    Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod
    tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam,
    quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo
    consequat. Duis aute irure dolor in reprehenderit in voluptate velit esse
    cillum dolore eu fugiat nulla pariatur. Excepteur sint occaecat cupidatat non
    proident, sunt in culpa qui officia deserunt mollit anim id est laborum.
  </p>
</section>

4 - Le CSS:

Dans ce CSS, je crée une class .menu, cette class est en position fixed, et positionnée de sorte à ce quelle sorte du viewport (partie visible de la page) avec translateX(100%). Le menu est donc caché.

Ensuite je crée une class menu-open qui sera rajouter sur le menu lorsqu'on veut qu'il s'affiche. Cette class positionne le menu pour qu'il soit visible (translateX(0%))

.menu {
    position: fixed;
    top: 0;
    right: 0;
    bottom: 0;
    left: 0;
    background-color: #fafafa;
    opacity: .9;
    transform: translateX(100%); /* je fais sortir l'élément du viewport */
    transition: .5s transform ease; /* animation */
}

.menu-open {
    transform: translateX(0%); /* je ramène l'élément dans le viewport */
    transition: .5s transform ease; /* animation */
}

.menu-btn {
  position: fixed;
  top: 20px;
  right: 20px;
  z-index: 1;
  padding: 2px 10px;
  background-color: #76DAFF;
  color: #fff;
  cursor: pointer;
}

5 - Le Javascript :

$(document).ready(function() {
    // je selectionne le bouton pour ouvrir le menu
    var btn = $(".menu-btn");
    // je selectionne le menu
    var menu = $(".menu");

    // je créer la function qui ouvre/ferme le menu
    var toggleMenu = function() {
        // j'ajoute ou enlève le class menu-open sur le menu
        menu.toggleClass("menu-open");
    };

    // J'écoute l'évènement click sur le btn
    btn.on("click", toggleMenu);
});
